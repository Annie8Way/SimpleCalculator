# SimpleCalculator 
### Простейший калькулятор, написанный на языке C# с использованием интерфейса Windows Forms. 

Содержит функции: 

* сложение 
* вычитание 
* умножение 
* деление 
* извлечение квадратного корня 
* смена знака
